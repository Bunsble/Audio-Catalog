# Организатор на личен аудио каталог

Конзолно Java приложение за управление на личен аудио каталог.

## Описание

Приложението позволява управление на различни типове аудио съдържание:
- Песни
- Музикални албуми
- Podcast-и
- Аудиокниги
- Playlist-и

Всеки обект има следните атрибути:
- Жанр
- Продължителност
- Категория
- Автор
- Година на публикуване

## Функционалности

- ✅ Добавяне на нов обект
- ✅ Изтриване на обект
- ✅ Търсене по заглавие, изпълнител, жанр и т.н.
- ✅ Създаване на playlist
- ✅ Добавяне и триене на обекти от playlist
- ✅ Връщане на информация за playlist, песен, албум и т.н.
- ✅ Филтрация по жанр, автор, година и т.н.
- ✅ Сортиране по азбучен ред
- ✅ Записване и четене на каталога във файл
- ✅ Записване и четене на отделни playlist-и във файл

## Компилиране и стартиране

### Компилиране:
```bash
javac src/*.java
```

### Стартиране:
```bash
java -cp src Main
```

## Структура на проекта

```
src/
├── AudioItem.java      - Базов клас за аудио обекти
├── Song.java           - Клас за песни
├── Album.java          - Клас за албуми
├── Podcast.java        - Клас за podcast-и
├── Audiobook.java      - Клас за аудиокниги
├── Playlist.java       - Клас за плейлисти
├── Catalog.java        - Клас за управление на каталога
├── FileManager.java    - Клас за работа с файлове
└── Main.java           - Главен клас с конзолно меню
```
